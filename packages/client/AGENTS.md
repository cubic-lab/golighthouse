# AGENTS.md

## Build Commands
- `bun build`: Builds the application
- `bun build-only`: Builds only the output files
- `bun type-check`: Runs TypeScript type checking

## Development
- `bun dev`: Starts development server
- `bun preview`: Starts preview server

## Code Style
- **Imports**: Use absolute imports where possible
- **Formatting**: Follow Prettier conventions
- **Types**: Use TypeScript for all code
- **Naming**: Use camelCase for variables, PascalCase for classes
- **Error Handling**: Always handle errors with try/catch blocks
- **Imports**: Group related imports together
- **Comments**: Add comments for complex logic

## TypeScript Configuration
- Uses Vue TypeScript configuration
- Includes all Vue files in src directory
- Uses ESNext module system

## Dependencies
- Vue 3
- TypeScript
- Vite
- PrimeVue

## Testing Framework
- Uses Vitest for testing
- Supports coverage reporting
